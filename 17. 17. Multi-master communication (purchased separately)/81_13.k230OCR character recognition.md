## k230 and MSPM0 communication

### 1. Experimental Prerequisites

This tutorial uses the MSPMG3507 development board, and the corresponding routine path is [14.export\MSPM0-K230\13_k230_ocr_rec].

K230 needs to run the [14.export\CanmvIDE-K230\13.ocr_rec.py] program to start the experiment. It is recommended to download it as an offline program.

 

Things you need:

Windows computer
MSPMG3507 development board
K230 visual module (including TF card with image burned in)
