# 0.Get start

The K230 vision module can be powered by a Type-C data cable or a communication interface, with a power supply voltage of 5V. The operating current of the deluxe version (with screen) when running an AI routine is about 0.8A, and the operating current of the standard version (without screen) when running an AI routine is about 0.6A. Due to the high power consumption, it is recommended to select the USB3.0 interface when connecting to a computer, or you can connect a 5V1A, 5V2A or 5V3A mobile phone adapter through a Type-C data cable for power supply.
